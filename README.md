# Time Series Analysis: Trend, Seasonality & Residual Decomposition (R)

## Overview
This project applies statistical time series techniques in R to analyse monthly sales data. The objective is to decompose the series into its core components—trend, seasonality, and residual noise—and assess model adequacy using diagnostic tools.

---

## Objective
The aim of this analysis is to:
- Identify long-term trends in the data
- Extract seasonal patterns (period = 12)
- Evaluate model fit using residual diagnostics
- Assess whether remaining variation behaves like white noise

---

## Dataset
The dataset consists of a single time series representing monthly observed values of a process over 504 time points.

---

## Tools Used
- R 
- Time series plotting functions (`ts.plot`)  
- Linear regression (`lm`)  
- Moving average smoothing techniques  
- Statistical diagnostics (histograms, QQ plots, differencing)
